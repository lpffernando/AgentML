# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-26

### Added
- Initial release
- Multi-agent AutoML framework built on OpenCode platform
- 7 specialized agents: coordinator, data-processor, feature-engineer, model-selector, model-validator, explainability, viz-agent
- 8 standardized skills: eda, data-cleaning, data-visualization, geospatial-analysis, feature-engineering, model-training, model-validation, shap-analysis
- MCP integration for python-executor and atlas-gis-tools
